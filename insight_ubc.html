Introduction and Video



Insight UBC is a platform designed for managing historical data related to university courses and buildings. It facilitates user actions such as uploading, querying, and removing datasets seamlessly. The platform allows users to access insightful statistics over time, presented through interactive graphs.

Throughout the development of Insight UBC, we incorporated several fundamental software engineering concepts:
  • Test-Driven Development (TDD): Using the Mocha and Chai frameworks, we wrote tests according to the specifications to ensure a smooth development process.
  • REST APIs: Using RESTful principles, we designed and implemented APIs that enables easy communication between the back-end server and the front-end user interface.
  • Agile and Scrum: Following Agile methodologies, particularly Scrum framework, we organized our development process into iterative sprints throughout the term, holding scrum meetings with a teaching assistant weekly.
